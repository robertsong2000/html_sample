# 🎨 互动教育演示项目集

这是一个专门为小朋友设计的互动教育演示项目集合，包含多个生动有趣的HTML页面，通过动画和交互的方式帮助孩子们学习科学、历史和技术知识。

## 📋 项目列表

### 1. 🏺 青铜器制作科普
**文件**: `bronze-vessel-making.html`
- **主题**: 古代青铜器制作工艺
- **特色**: 分步骤展示青铜鼎的制作过程
- **互动**: 点击步骤查看详细动画，支持自动演示模式
- **知识点**: 陶范制作、青铜熔化、浇铸工艺、雕刻装饰

### 2. 🏺 青铜器制作科普 - 增强版
**文件**: `bronze-vessel-enhanced.html`
- **主题**: 升级版青铜器制作演示
- **特色**: 更丰富的动画效果，包括工匠工作、火焰熔炉、火花飞溅
- **互动**: 完整的6步制作流程，支持手动和自动模式
- **视觉效果**: 动态材质、粒子效果、工作场景

### 3. 🔍 3D青铜器探秘
**文件**: `bronze-vessel-3d.html`
- **主题**: 3D技术展示青铜器
- **特色**: Three.js驱动的3D模型查看器
- **互动**: 360度旋转、缩放、平移，支持加载自定义3D模型
- **功能**: 顶点数、面数统计，模型信息展示

### 4. ⚙️ 齿轮积木科普
**文件**: `gear-learning.html`
- **主题**: 机械传动原理
- **特色**: 可拖拽的齿轮积木系统
- **互动**: 自由组合齿轮，观察传动效果
- **挑战模式**: 创建特定传动比的齿轮系统

### 5. 🌑 日蚀动画
**文件**: `eclipse-animation.html`
- **主题**: 日蚀天文现象
- **特色**: Apple风格设计，精美动画效果
- **互动**: 播放控制、速度调节、阶段追踪
- **视觉效果**: 日冕、星空背景、动态光影

### 6. 🌙 月食科普
**文件**: `lunar-eclipse-education.html`
- **主题**: 月食形成原理
- **特色**: 太阳系模型演示
- **互动**: 月食模式切换、月相追踪
- **教育内容**: 月食阶段、科学解释、趣味知识

## 🚀 使用方法

1. **直接打开**: 双击任意HTML文件即可在浏览器中运行
2. **本地服务器**: 使用Python或其他工具启动本地服务器
   ```bash
   python -m http.server 8000
   ```
3. **在线访问**: 将文件上传到网络服务器即可在线访问
